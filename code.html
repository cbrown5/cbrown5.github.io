About Our R Courses

We provide practical courses in R programming focused on ecological and environmental data analysis. These courses are designed for scientists, students, and practitioners working with environmental data.

Available Courses

Introduction to R for Ecologists

Learn the basics of R programming with ecological examples. This course covers data import, basic statistics, and creating publication-quality graphs.

Data Wrangling in R

Master data manipulation techniques using tidyverse packages. Learn to clean, transform, and prepare ecological data for analysis.

Spatial Analysis in R

Work with spatial data in R. Learn to create maps, analyze spatial patterns, and work with GPS data.

Statistical Modeling

Learn to build and interpret statistical models for ecological data, including GLMs, GAMs, and mixed effects models.
